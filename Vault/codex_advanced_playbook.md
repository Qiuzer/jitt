# Codex 工程级操作手册（进阶版）

## 一、核心升级认知

-   你不是在写代码，而是在管理 Agent
-   Codex 是可执行智能体（读代码 / 改文件 / 跑命令）

------------------------------------------------------------------------

## 二、Plan / Ask → Code 工作流

### Step 1：Ask

请先给出实现方案： - 架构设计 - 涉及文件 - 风险点

### Step 2：确认方案

### Step 3：执行代码

------------------------------------------------------------------------

## 三、Prompt 工程

### 四要素

1.  目标
2.  上下文
3.  约束
4.  验收标准

------------------------------------------------------------------------

## 四、上下文管理

-   只提供必要代码
-   避免上下文过长

------------------------------------------------------------------------

## 五、AGENTS.md

# AGENTS.md

## 十一、常见错误

-   不做计划
-   上下文过多
-   频繁 steer
-   无验收标准

定义 → 计划 → 执行 → 审查 → 重构 → 自动化
